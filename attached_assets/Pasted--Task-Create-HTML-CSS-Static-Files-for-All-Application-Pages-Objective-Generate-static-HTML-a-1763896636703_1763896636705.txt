HTML File Requirements:
Semantic HTML5: Use proper semantic tags (header, nav, section, aside, main, footer)
Accessibility: Include ARIA labels, alt text for images, semantic structure
Mobile-First: Start with mobile then scale to desktop
No JavaScript: Pure static HTML files (no functionality, just structure)
Responsive Images: Use placeholder images with proper aspect ratios
Meta Tags: Include viewport, charset, title, description
CSS File Requirements:
Desktop Breakpoint: 1024px+ (or custom)
Mobile Breakpoint: Below 768px
Tailwind-Compatible Classes: Use class names similar to Tailwind (flex, gap, p-4, text-lg, etc.) OR pure CSS with CSS variables
Color Variables: Define in assets/colors.css with Vietnamese naming
Responsive Design: Use CSS Grid/Flexbox for responsive layouts
No Framework Dependencies: Pure CSS (no Bootstrap, no Tailwind import)
Dark Mode Support: Include both light and dark theme CSS variables
